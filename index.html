Acerca de Datasnap

Datasnap integra actualmente diversas funcionalidades clave orientadas a la conversión, análisis y optimización de datos, entre las cuales se encuentran:

Lectura de Archivos

Puede procesar archivos csv, txt, json y sql.

Corrección de Errores

Eliminación de datos repetidos y optimización.

Copias de Seguridad

Cada vez que se modifique se guarda una copia de respaldo.

Estadísticas

Se puede analizar los datos del archivo y otorga un informe de estadísticas.
